# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [Unreleased]

### Security
- Upgraded `urllib3` 2.6.3 → 2.7.0 to patch CVE-2026-44431 (sensitive header
  leak on cross-origin redirects) and CVE-2026-44432 (decompression bomb via
  streaming Brotli / `drain_conn()`)
- Fixed path traversal in `--filename` argument: strip directory components
  with `Path.name` before joining into `~/Downloads`
- Added `bandit` SAST to CI pipeline to catch security issues automatically
- Enabled GitHub secret scanning and push protection on the repository
- Added CodeQL SAST workflow (runs on push/PR and weekly)

### Changed
- Switched dependency management to [pip-tools](https://pip-tools.readthedocs.io/):
  `requirements.in` is the source of truth; `requirements.txt` is fully pinned
  and autogenerated via `pip-compile`
- Added explicit `urllib3>=2.7.0` floor in `requirements.in` to prevent
  future `pip-compile` runs from resolving a vulnerable version
- `--model` argument now validates against `["dall-e-3", "dall-e-2"]` via
  argparse `choices`, rejecting arbitrary values at parse time
- HTTP error output now prints only the API error message from the JSON
  response body instead of the full raw response text
- CI matrix expanded to Python 3.12 and 3.13
- CI upgraded to `actions/checkout@v6` and `actions/setup-python@v6`
- Added `pip-audit` dependency vulnerability check to CI

## [1.0.0] - 2024-03-01

### Added
- Initial release: DALL-E 3 image generation via OpenAI API
- CLI flags: `--prompt`, `--model`, `--number`, `--size`, `--quality`,
  `--style`, `--output`, `--download`, `--filename`
- Input validation for `size`, `quality`, and `style` arguments
- Automatic download to `~/Downloads` with optional custom filename
